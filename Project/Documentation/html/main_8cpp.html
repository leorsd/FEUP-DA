int main ( )

Main function of the program. It loads the graph from the CSV files, asks the user for the mode of operation, and runs the selected mode.

The program will ask the user for the mode of operation, either batch mode or menu mode. In batch mode, the program will read from a input.txt file, while in menu mode, the user will be prompted to enter the data manually. The program will then call the appropriate function to run the selected mode.

Returns
int Returns 0 if the program runs successfully.
