Detailed Description

A class representing a mutable priority queue.

This class implements a mutable priority queue using a binary heap. It provides efficient operations for inserting elements, extracting the minimum element, and decreasing the key of an element. It is used in Dijkstra's algorithm to manage vertices based on their distances from the source node.
